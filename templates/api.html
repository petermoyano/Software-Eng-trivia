This site communicates with the quizapi.io api wich is a simple REST api that requires an API_KEY to operate.

The purpose of this API is to provide a platform for websites to embed quick quizzes within their products, thus providing one additional feature to the user.

When a user submits the parameters of the quiz he or she wants to take, Python's requests built in library makes a GET request to the QuizAPI with the API_key for authentication. Once the JSON response is received, the quiz is rendered to the user. All additional functionality, like checking for right answers and show how the user did is done in the back-end through Python and Jinja.
